## Tenants

### `client.tenants.create(data)`
Create a new tenant.
```typescript
const tenant = await client.tenants.create({
  name: "Acme Corp",
  external_id: "cust_12345", // optional
});
```

### `client.tenants.list(params?)`
List all tenants.
```typescript
const { data, total } = await client.tenants.list({
  status: "connected", // optional filter
  limit: 50,
  offset: 0,
});
```

### `client.tenants.get(tenantId)`
Get a single tenant.
```typescript
const tenant = await client.tenants.get("tenant_uuid");
```

### `client.tenants.delete(tenantId)`
Delete a tenant and all associated data.
```typescript
await client.tenants.delete("tenant_uuid");
```

### `client.tenants.createSetupLink(tenantId, options?)`
Generate a WhatsApp connection link.
```typescript
const link = await client.tenants.createSetupLink("tenant_uuid", {
  success_redirect: "https://yourapp.com/connected",
  failure_redirect: "https://yourapp.com/error",
});
console.log(link.url); // Share with your customer
```

### `client.tenants.getInboxToken(tenantId)`
Generate a JWT for the inbox embed.
```typescript
const { token, expires_at } = await client.tenants.getInboxToken("tenant_uuid");
```

## Messages

### `client.messages.send(tenantId, data)`
Send a message.
```typescript
const msg = await client.messages.send("tenant_uuid", {
  to: "+250788000001",
  type: "text",
  content: { body: "Hello!" },
  idempotency_key: "unique-key", // optional
});
```

### `client.messages.list(tenantId, params?)`
List messages.
```typescript
const { data, has_more } = await client.messages.list("tenant_uuid", {
  conversation_id: "conv_uuid", // optional
  direction: "inbound",         // optional
  status: "delivered",          // optional
  limit: 50,
  before: "2026-03-27T00:00:00Z", // cursor pagination
});
```

### `client.messages.markAsRead(tenantId, messageId)`
Send a read receipt.
```typescript
await client.messages.markAsRead("tenant_uuid", "message_uuid");
```

